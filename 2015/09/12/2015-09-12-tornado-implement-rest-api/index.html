这里主要来实现简单的Basic Auth，
登陆时传username， password，登陆成功之后会返回用户信息，并在用户信息中加一个token字段，以后用户请求用户相关的数据时，将这个token放在http头传至服务器，登陆失败当然是401错误.
basic auth的token就是简单的用户名和密码进行base64加密，(即:base64.b64encode(“username:password”.encode()))
